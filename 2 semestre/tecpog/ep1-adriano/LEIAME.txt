DESCRIÇÃO:
Este programa tem o objetivo de buscar primos. Ele opera em dois modos:
No modo 0, dado um número, busca pelo primeiro primo imediatamente após este número.
No modo 1, dado um número, busca se é possível decompor este número como uma multiplicação de dois primos, imprimindo-os caso achar, ou não imprimindo nada caso contrário.
É possível utilizar números de 1 a 999999936 como entrada.



COMO COMPILAR:
Para compilar o programa em assbembly, execute os seguintes comandos no terminal (com o diretório onde o programa está, aberto):

	nasm -f elf64 ep1.s
	ld -s fonte.o

Isso irá gerar o executável do programa, que será executado



COMO EXECUTAR:
Para executar o programa, use no terminal o comando:

	./ep1

Ao executar o programa, o usuário deve digitar duas entradas, separadas por espaço:
A primeira como 0 ou 1, condizente com o modo desejado (os modos estão explicados na descrição).
A segunda com o número com o qual se deseja fazer a operação.
